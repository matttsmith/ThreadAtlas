You are classifying individual messages from an AI chat conversation.
For each message, assign a register and reality_mode.

REGISTER values (pick exactly one):
- work: professional tasks, coding, engineering, business operations
- research: learning, academic inquiry, fact-finding
- career_logistics: job search, resume, interviews, career planning
- creative_writing: user asked the AI to write fiction, poetry, scripts
- roleplay: user and AI are acting as characters in a scenario
- jailbreak_experiment: user is testing AI limits, prompt injection, bypasses
- puzzle_hobby: games, riddles, trivia, recreational problem-solving
- personal: health, relationships, life advice, therapy-adjacent
- other: does not fit any of the above

REALITY_MODE values (pick exactly one):
- literal: the speaker means what they say directly
- fictional: the speaker is writing or performing as a character
- hypothetical: the speaker is exploring a what-if scenario

CRITICAL DISTINCTIONS:
- A user saying "Write a story where a villain says 'I will destroy everything'" is register=creative_writing, reality_mode=literal (the USER is literally asking for creative output).
- A user saying "I am the Dark Lord. I will destroy everything!" inside a roleplay session is register=roleplay, reality_mode=fictional.
- A user saying "I decided to switch to Python for the backend" is register=work, reality_mode=literal.
- An assistant replying with "I will implement that for you" is register=work, reality_mode=literal.
- A user saying "What if we used Rust instead?" is register=work, reality_mode=hypothetical.
